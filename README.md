# csv-quiz-practice
# 刷题啊

基于 CSV 文件的 Obsidian 刷题插件。将你的题库写入 CSV，然后在 Obsidian 中刷题、标记、筛选、统计正确率。

## 快速开始

1. 在 vault 根目录创建 `题库.csv`（也可在设置中指定其他路径）
2. 点击左侧 ribbon 的图书图标，或执行命令 `打开刷题面板`
3. 开始刷题

## CSV 格式

CSV 文件必须为 **BOM 前缀的 UTF-8** 编码，15 列：

```
序号,题干,选项A,选项B,选项C,选项D,正确答案,标签,一级分类,二级分类,三级分类,收藏,掌握,重复,错题
```

- **序号**：题目的唯一标识，必须是数字
- **题干**：支持 Markdown 渲染（可包含 Obsidian 内部链接）
- **选项A~D**：四个选项的内容
- **正确答案**：如 `A`、`B`、`C`、`D`
- **标签**：空格分隔的多个标签，如 `#数学 #代数 #高考`
- **一级分类 / 二级分类 / 三级分类**：三级分类体系，用于筛选
- **收藏 / 掌握 / 重复 / 错题**：布尔标记，值为 `1`（是）或 `0`（否），刷题过程中自动更新

### 示例行

```
1,1+1=?,1,2,3,4,B,#数学 #加法,#数学,,,1,,,
```

## 功能

### 刷题

- 选择答案后实时反馈对错，显示正确答案
- **答对自动跳转**：可设置答对后延迟秒数自动进入下一题（设置中调整）
- **随机题目顺序**：每次加载时重新排列题目
- **随机选项顺序**：每题选项随机排列

### 筛选

- **标签筛选**：点击标签芯片切换筛选，支持多标签交集（多个标签同时匹配）
- **分类筛选**：三级分类联动下拉筛选
- **标记筛选**：收藏、掌握、重复、错题三个状态切换（不限 / 仅 / 否）
- 筛选结果自动更新题目列表和进度

### 编辑

- 答题过程中可编辑当前题目的标签、分类和标记
- 修改自动保存到 CSV 文件

### 进度持久化

- 答题记录、正确率、当前进度自动保存
- 关闭 Obsidian 后重新打开，自动恢复到上次位置
- 切换 CSV 文件路径会自动开启新的刷题会话

### 设置

| 设置项 | 说明 |
|---|---|
| CSV 文件路径 | 题库 CSV 文件路径（相对于 vault 根目录） |
| 随机题目顺序 | 开启后每次加载随机排列题目 |
| 随机选项顺序 | 开启后每题选项顺序随机 |
| 答对自动跳转延迟 | 答对后自动跳转的等待秒数（0 = 不自动跳转） |
| 默认展开筛选栏 | 打开面板时筛选栏默认展开/折叠 |
| 默认展开编辑栏 | 打开面板时标签编辑栏默认展开/折叠 |
| 标记筛选默认值 | 各标记筛选（收藏/掌握/重复/错题）的默认状态 |
| 重置刷题进度 | 清除所有答题记录、统计和筛选状态，重新加载题库 |
